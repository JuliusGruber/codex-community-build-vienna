# Repository Instructions

## Git Workflow

- Always commit completed changes and push them directly to `main`.
- Do not create feature branches.
- Do not commit unrelated changes.
- Use concise commit messages that describe the completed change.

## Links

- When asked to add a link, add it to the root `links.md`.
- Use the destination hostname as the link label (for example, `developers.openai.com`) so the destination site is clear.
- Include a short description for every link.
